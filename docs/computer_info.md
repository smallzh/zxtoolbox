# 获取计算机的硬件信息

跨平台，支持 Windows、Mac、Linux 操作系统，获取各个平台下的硬件基础信息，包括 CPU、GPU、内存、硬盘、网络信息、操作系统信息。各种信息以 Table 表格的形式展示。

## 0x01. 命令格式

```bash
zxtool ci [选项]
```

| 选项 | 说明 |
|------|------|
| （无选项） | 显示简短汇总信息（默认） |
| `-a`, `--all` | 显示详细信息 |

## 0x02. 输出简短汇总信息

```bash
zxtool ci
```

输出示例：

```
+--------+-------------------+
| 类别   | 详情              |
+--------+-------------------+
| OS     | Ubuntu 24.04      |
| CPU    | 8 Cores, 3.0 GHz  |
| GPU    | NVIDIA A10        |
| Memory | 32 GB             |
| Disk   | 512 GB            |
| Network| 192.168.1.12     |
+--------+-------------------+
```

## 0x03. 输出详细信息

```bash
zxtool ci --all
```

显示每个硬件类别的详细信息表格，包括 CPU 型号与频率、GPU 显存、内存使用情况、各磁盘分区详情、网络接口列表等。
