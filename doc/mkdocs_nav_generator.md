# Mkdocs目录自动生成功能

基于Mkdocs项目当前的`mkdocs.yml`中的 **nav** 节点 和 文档目录 `doc` 自动生成 整个的 nav节点

## 1. 生成导航节点

功能描述：根据nav节点和 doc目录里的 md文件的层次结构，生成 nav 导航内容

命令行: `zxtoolbox -m -d docs`，参数说明如下:

| 名称 | 说明                                                          |
|----|-------------------------------------------------------------|
| -m, --mkdocs  | 激活 mkdocs 相关功能                                                |
| -d, --doc  | mkdown文档的存储目录名称                                                      |
