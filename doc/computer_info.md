# 获取计算机的硬件信息

跨平台，支持Window、 Mac、 Linux操作系统，获取各个 平台下的 硬件基础信息，包括 CPU、 GPU、内存、 硬盘、网络信息、 操作系统信息。 各种信息以 Table表格的 形式展示


## 1. 输出整体的 简约汇总信息

功能描述：输出操作系统和硬件的简约 Table形式的信息

命令行: `zxtoolbox -c -s`，参数说明如下:

| 名称 | 说明                                                          |
|----|-------------------------------------------------------------|
| c  | 激活 计算机信息显示功能                                                |
| s  | 打印简短信息                                                      |

打印格式如下：

| 类别 | 详情           |
|----|--------------|
| OS | Ubuntu24     |
|CPU| 8Cores, 3GHz |
|GPU| Nvidia A10   |
|Memory| 32G          |
|Disk| 512G         |
|Network| 192.168.1.12 |

## 2. 输出整体的 详细信息

功能描述：尽可能的输出操作系统和硬件的详细信息，以Table形式显示

命令行： `zxtoolbox -c -a`， 参数说明如下：

| 名称 | 说明           |
|----|--------------|
| c  | 激活 计算机信息显示功能 |
| a  | 打印尽可能多的详细信息  |
