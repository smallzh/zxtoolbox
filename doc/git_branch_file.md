# Git分支文件汇总

基于git工具

## 1. 导出当前分支下的 文件信息

功能描述: 导出 当前分支下，除`.gitignore`文件忽略的，其余全部文件信息

命令行: `zxtoolbox -g -s -i include.txt -o ../branch_files`，参数说明如下:

| 名称 | 说明                                                          |
|----|-------------------------------------------------------------|
| g  | 激活 git分支相关导出功能                                              |
| s  | 激活 文件导出功能                                                   |
| i  | 包含检测的文件，为 txt文件，支持通配符，每行 表示一类文件，比如 `*.java`表示 包含全部的 .java文件 |
| o  | 指定最终文件的生成目录                                                 |

要求： 假如当前在`master`分支，通过运行以上命令 ，文件信息全部输出到一个 json文件中, 文件的名称格式为 `${project_name}-${branch_name}-files`，其中json 内容如下:

```json
{
  "name": "master",
  "export_time": "2026-02-05 10:33:50",
  "project_path": "d:/project/xxx",
  "files": [
    {
      "fid": "xxxx",
      "path": "docs/index.md",
      "size": 3000,
      "md5": "文件的md5值"
    },
    {
      "fid": "yyyy",
      "path": "docs/intro.md",
      "size": 320,
      "md5": "文件的md5值"
    }
  ]
}
```

其中各个参数的含义：

| 名称           | 说明                               |
|--------------|----------------------------------|
| name         | 分支名称，比如master                    |
| export_time  | 导出的时间，格式为: `YYYY-MM-dd HH:mm:ss` |
| project_path | 项目的绝对路径                          |
| files.fid    | 文件的唯一标识                          |
| files.path   | 文件的相对路径                          |
| files.size   | 文件的大小                            |
| files.md5    | 文件的md5值                          |

关于文件名中的 ${project_name}，表示项目名称，${branch_name}表示 git的 当前分支名称。


## 2. 导出分支下的 差异文件

功能描述：基于 前面1 中 导出的 json文件，和当前分支的文件进行比对，比如 当前分支为 `bug-1`，导出 当前分支的 全部差异文件，包含新增的、修改的文文件，最终输出包含2部分：

1. 差异文件信息：以json格式存储，内容同 前面1中的json内容一致
2. 不一致的差异文件本身：文件所处的目录层级，以 下划线(_) 来连接，最终 作为 文件的 名称

命令行：`zxtoolbox -g -e -f p_b_files.json -o ../out`， 其中命令行参数说明如下：

| 名称 | 说明             |
|----|----------------|
| g  | 激活 git分支相关导出功能 |
| e  | 激活 文件差异导出功能    |
| f  | 待对比的文件路径       |
| o  | 指定最终文件的生成目录    |


## 3. 复制差异文件到对应目录

功能描述：将 功能2种导出的 差异文件，按照对应的json描述文件 复制到对应的目录中

命令行：`zxtoolbox -g -c -f p_b_files.json`，其中命令行参数说明如下：

| 名称 | 说明            |
|----|---------------|
| g  | 激活 git分支相关导出功能 |
| c  | 激活 差异文件复制功能   |
| f  | 参考的文件路径       |
